Product Catalog

Product Catalog for Odoo

This Odoo module extends the product catalog feature to Inventory Transfers, offering a user-friendly, POS-style product selection interface to improve speed and accuracy in product entry.

Effortless Integration, Enhanced Accuracy

Odoo natively provides a Product Catalog feature for apps like Sales, Purchases, Manufacturing, and Repairs. This catalog presents products in a visual, card-style interface, enabling users to search, filter, and select products efficiently.

This module brings the same experience to the Inventory modules by integrating a Catalog button into the Transfer Operations tabs. Users can browse all available products, view product details (image, reference, quantity, attributes), and quickly select items for entry, mimicking a retail-like point-of-sale interaction.

Detailed Features

01 POS-style catalog view: A visually intuitive, POS-style product catalog where each product is shown as a card with its image, name, price or cost, internal reference, and available quantity-making selection faster and more user-friendly.
02 Inventory integration: Integrates the product catalog into stock pickings, enabling users to visually select products for incoming, outgoing, or internal transfers with real-time quantity controls.
03 Product filters and search: Quickly locate products using a powerful search and filtering system—search by name, barcode, or reference, and filter by category, product type, or variant attributes for precise selection.
04 Incremental selection: Easily select products with incremental adjustments—click to add one unit or use the + / − controls to increase or decrease the quantity as needed.
05 Seamless return: After selecting products from the catalog, they are seamlessly returned and automatically added to the original source form, ensuring smooth workflow and reducing manual entry.
06 Minimal setup required: The module requires minimal setup and is fully integrated with Odoo's standard product and UI logic, ensuring easy implementation and compatibility with existing workflows.

Set Up

To begin with, have the Product Catalog Module downloaded on your system. For its proper setup, go through the steps below.

Firstly, you need to install the Product Catalog App from Odoo Apps. It's easy to locate, just search for the name, you can identify it with the product catalog icon. Ta-da, you are ready to go!!

Apps >> Product Catalog >> Activate

Work Flow

For creating a Transfers Go to Inventory > Operations > Transfers.
Create a new transfer or open an existing one.
In the Operations tab, click the Catalog button.
You can see the kanaban view of products. Browse and add products as described above.
Click Back to Quotation to return and review the line items.
You can see the selected products are added inside the transfer.
The Product Catalog module brings the power and simplicity of the visual product catalog into core operational workflows, including invoicing and stock transfers. It enhances user productivity, improves accuracy, and delivers a unified experience across more Odoo apps.
