# Zencore Apps Dashboard

Odoo 18 addon that adds a customizable apps dashboard experience inspired by the supplied reference screenshots.

Technical module name: `zencore_apps_dashboard`

Main features:

- User-level dashboard customizer toggle in Preferences.
- Clock and greeting on the apps dashboard.
- Apps search filter.
- Appearance popup for icon size, text size, corner radius, layout width, glassmorphism, animations, clock/search visibility, background opacity, and background image.
- Settings are stored on `res.users`.
