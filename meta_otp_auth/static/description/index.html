Two layer Authentication

Two layer Authentication:An impressive application which adds extra layer of security and provides an option to log-in/sign-up via OTP.

Features:

  • Functionality to enable OTP during sign-up
  • Functionality to enable OTP during sign-in
  • Flexibility to login via OTP or password.
  • Validates valid email during sign-up
  • The OTP validation information along with their success status will be maintained in the backend.
  • Check existing user during sign-up
  • An OTP is sent to a registered email address and the user will only be able to register/login on entering the same OTP.
  • Admin can set expiry time and email template of OTP.
  • Admin can set OTP type (password or text).
  • Cross browser compatible.
