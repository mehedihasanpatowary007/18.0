Sales Report Filter Grouping & Status

Advanced Sales & POS reporting with powerful filtering, grouping, and status management

📊 Overview

Transform your sales and Point of Sale reporting with this comprehensive module that adds advanced filtering, intelligent grouping, and real-time status tracking to both Sales Orders and POS Orders in Odoo.

🎯 Perfect for: Retail businesses, wholesale operations, multi-channel sales, and businesses using delivery courier services.

✨ Key Features

🎯 Advanced Filtering

  • Report Category (Retail, Wholesale, Online, Direct Sales)
  • Priority Levels (Low, Medium, High, Urgent)
  • Sales Region tracking (North, South, East, West, Central, International)
  • Automatic Customer Segmentation (New, Regular, VIP, Inactive)
  • Point of Sale assignment

📈 Status Tracking

  • Vape Status workflow (Quotation → Sales Order → Send to Courier → Done)
  • Overall Status (combining sale, payment, and delivery)
  • Payment Status (Pending, Partial, Paid, Overdue)
  • Delivery Status (Not Shipped, Partially Shipped, Shipped, Delivered)
  • Order Progress indicators with visual bars
  • Line-level status management

📑 Smart Grouping

  • Group by Vape Status
  • Group by Overall Status
  • Group by Point of Sale
  • Group by Category & Region
  • Group by Customer Segment
  • Group by Payment/Delivery Status
  • Group by Session State (POS)

🛍️ POS Enhancements

  • Payment Status tracking for POS orders
  • Session State tracking
  • Customer Segmentation for POS customers
  • Order Age tracking
  • Margin calculation
  • POS Report Analysis (Pivot & Graph views)

📊 Analytics & Insights

  • Order Age tracking
  • Days to Confirm metrics
  • Fulfillment Progress tracking
  • POS Report Analysis with pivot tables and charts
  • Revenue and margin tracking
  • Enhanced Sale Report with POS and Vape Status filters

🚚 Courier Integration

  • Integrates with delivery_courier_custom module
  • Automatic status updates when courier assigned
  • Real-time tracking across sales and delivery
  • Email notifications
  • Works standalone with standard Odoo too

🔄 Vape Status Workflow

Automatic workflow tracking for sales orders with courier integration:

1. Quotation - Order is in draft or sent state
2. Sales Order - Order is confirmed, delivery order created
3. Send to Courier - Automatically updated when delivery assigned to courier
4. Done (Delivered) - Automatically updated when delivery is completed

Benefits: Real-time status updates, no manual updates needed, complete visibility, integrated email notifications.

🚀 Quick Start

For Sales Orders:

  1. Install the module from Apps menu
  2. Navigate to Sales → Orders → Quotations/Orders
  3. Open any order and fill in the new fields in "Other Information" tab:
    • Point of Sale (if applicable)
    • Report Category
    • Priority Level
    • Sales Region
    • Delivery Status
  4. Use advanced filters and grouping options in the search bar
  5. View enhanced reports at Sales → Reporting → Sales

For POS Orders:

  1. Navigate to Point of Sale → Orders
  2. View enhanced columns for priority, payment status, customer segment
  3. Use new filters: High/Urgent Priority, Payment Status, VIP/Regular Customers
  4. Group by: Point of Sale, Category, Region, Segment, Payment Status, Session State
  5. Access POS Report Analysis for pivot and graph views
